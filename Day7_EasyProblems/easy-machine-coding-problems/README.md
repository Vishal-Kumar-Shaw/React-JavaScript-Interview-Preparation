🚀 Counter App — But “Interview Grade”

Instead of basic counter, build 3 levels:

🟢 Level 1 (Basic – Warmup)

Features:

Increment

Decrement

Reset

👉 Focus:

React: useState

Angular: Component + property binding + event binding

🟡 Level 2 (Intermediate – Interview Ready)

Features:

Step increment (e.g., +1, +5, +10)

Min/Max limit

Disable buttons at limits

Show history of actions

👉 Focus:

React: useReducer

Angular: Services + state sharing

🔴 Level 3 (Advanced – Stand Out Project)

Features:

Undo / Redo

Persist state (localStorage)

Multiple counters

Global state management

👉 Focus:

React: Context API / Redux

Angular: RxJS + BehaviorSubject / NgRx (optional)
